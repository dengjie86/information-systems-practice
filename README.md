# 校园宿舍报修与工单管理系统

## 项目简介
本项目是一个集“报修申请—工单流转—维修处理—评价反馈—数据统计”于一体的信息系统。旨在解决传统宿舍报修中信息丢失、进度不透明等问题，提高高校宿舍后勤服务效率。
