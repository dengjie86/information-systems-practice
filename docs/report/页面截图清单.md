## 六、截图操作约定

- 截图前先用 `database/mock_data.sql` 把演示数据灌进去，保证有看头
- 浏览器开 1440×900 或更高，**不要带书签栏**，截图区域裁掉浏览器地址栏
- 涉及个人信息的字段（手机号、姓名）用 mock 数据里的就行，不要拿真名
- 截图保存到 `docs/report/assets/<文件名>`，本目录的 README / 中期报告会用相对路径引用

## 七、引用方式

中期报告里如果要插图，写法：

```markdown
![学生端工单列表](./assets/03-student-list.png)
```

后续答辩 PPT 直接复用这些 PNG，不要再单独截一遍。
